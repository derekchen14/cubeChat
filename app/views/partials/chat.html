Technologies Used

Front-end: As a modern web app, the lines where one library ends and another framework begins really start to blur. Here is one possible breakdown.

  • AngularJS - By providing context for our data model, the controller allows us to establish data-binding between the model and the view
  • HTML5 - We connected the dots between the presentation, data, and logic components as follows Web Audio API. This also includes the Video API.
  • CSS3 - There are things like alphatransparency, border radius and semantic changes. Also, I declared the Doctype as HTML 5, so yea.
