Technologies Used

Front-end: As a modern web app, CubeChat uses many tools to make a complex operation look smooth and easy.

  • AngularJS - Custom built controllers, filters, directives, partials, route providers, services, factories, form validation with regex, and more!
  • HTML5 - Angular uses declarative mark-up. In addition, there is the Web Audio API, Video API, and Web Socket API through sockets.io library.
  • CSS3 - From the cube transition itself to the arrows that change color only when you click, everything to do with the cube design requires some pretty sophisticated CSS.

Back-End: A full-stack app wouldn't be complete without all the components running on the server as well.

  • NodeJS - The event-driven, single-threaded platform that makes the realtime action possible, turbo-boosted by the Express framework.
  • WebRTC - Using STUN, TURN and ICE for NAT Traversal, we go through different firewalls to establish a P2P connection.
  • MongoDB - NoSQL database connected to the app using Mongoose. Embedded documents are used for denormalization by creating a hierachical structure where chat messages belong to a user.
  • Yeoman - Probably belongs in a separate section. Yo for scaffolding, Grunt for task automation and Bower for package management.
